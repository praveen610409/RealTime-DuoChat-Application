# Real-Time Chat App

A simple two-user chat application built using HTML, CSS, and JavaScript.
Users can chat in the same browser window, with messages displayed on the left or right based on the sender.

## Features âœ…

#### Two chat boxes: User A and User B

#### Right side: Your own messages (green bubble)

#### Left side: Other user messages (blue bubble)

#### Scrollable chat area

#### Responsive design

#### Press Enter to send messages

#### Color-coded messages for clarity

#### Works without any backend

## Demo

Open index.html in your browser to try it out.
(No server or backend required.)
